Memory Matching Game

A simple Memory Matching Card Game built with HTML, CSS, and JavaScript. Players flip cards to find matching pairs, testing their memory and observation skills. The game includes basic sound effects for correct and incorrect matches.

Project Overview

The Memory Matching Game runs directly in the browser. Players click on cards to flip them and try to match all pairs. Correct matches stay open, while incorrect ones flip back, creating a simple and interactive gameplay experience.

Features

Interactive card flipping
Sound effects for matches and mismatches
Clean, responsive design
Runs directly in the browser
Simple and easy-to-understand JavaScript game logic

Technologies Used

HTML5 – Game structure
CSS3 – Styling and layout
JavaScript – Game logic and interactivity

Learning Objectives

By building this game, you will:

Understand DOM manipulation using JavaScript
Practice event handling for user interactions
Implement basic game logic
Improve problem-solving and debugging skills
Gain experience building a small front-end project

How to Play

Click on a card to flip it and reveal its symbol
Click on another card to try to find the matching pair
If the cards match, they stay open
If they do not match, they flip back after a short delay
Continue until all pairs are successfully matched
